# 杂项

> 全部命令需要OP权限/后台执行, []为必填, ()为选填

## help

> /ni help > 查看帮助信息

## reload

> /ni reload (重载类型) > 重新加载NI物品

* `(重载类型)` 重载哪一部分内容(不填则重载全部内容)

    支持的重载类型有:
    
    * `config` 重载插件配置
    * `item` 重载全局节点及物品配置
    * `script` 重载脚本节点
    * `pack` 重载物品包
    * `action` 重载物品动作
    * `editor` 重载物品编辑函数
    * `expansion` 重载JavaScript扩展
    * `其他` 填入其他参数视为重载所有内容
